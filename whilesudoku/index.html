While Sudoku is a series of sudoku puzzle books. The puzzles are created via Python programming. Each puzzle will have a unique pattern and solution which can be determined entirely by logic. Herein, you'll find daily-updated puzzles that can be printed or saved to your computer / device as an image.

Paperback book versions are released on Amazon. They are formatted with 2 puzzles per page, with 5.25" x 8" dimensions, a decent size for easier portability.

I also generate Digital (PDF) books of the same name at whilesudoku.gumroad.com. These are formatted for e-ink "paper" tablets (6.18" x 8.24"), or devices that allow on-screen note-taking. But since they are PDF's, they should be compatible with most, if not all, devices that can support opening PDF files.

Please visit my While Sudoku Facebook Page, and give it a like! Thank you!
