I like the Weather Climate Toolkit from NOAA. The above is a proposed and interactive vision of mine to remedy a weakness in it, namely: Conveniently viewing recent Level-III Radar data. The NWS Telecommunications Gateway is home to approximately ~200+ recent images × 150+ radar sites. So I'm hoping an extra tab like this could bridge the gap between current, frustrating usage for viewing L3 data and a much quicker way to visualize L3 data. Meanwhile, the generated URLs can be copy/pasted for use in WCT in either the 'Single File' tab or 'URL Directory' tab.

PROS: Much quicker/easier use of near-real-time L3 data.
CONS: Files are not in a standarized format (i.e., they are named sn.####, instead of "KMHX_SDUS52_N0RMHX_200309182020"), so WCT's "Product Overlay Viewer," which time-matches downloaded legacy products with downloaded supplementary files (like Storm-tracking, TVS, etc...), will not work unless you change the file names. WCT grants access to UCAR's THREDD's server which houses around a month of past radar data, but it's kind of clunky to maneuver with and their files are not the standardized file-naming either.
