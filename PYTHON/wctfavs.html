Python Creations - Radar .WCTFAVS Files

WCTFAVS_files_072018.zip

WCTFAVS files work exclusively in NOAA's Weather Climate Toolkit (WCT). WCTFAVS is a type of file extension for a basic-formatted text file. The favorites feature in WCT is typically used to store your favorite items, but I engineered these files with Python, specifically for use with the favorites feature. They specifically access the most recent Level-III radar data.

WCT makes accessing Level-II data very simple, but L3 data is a little more difficult. You can now access the data via a menu process, but to do that every time will get old. These files can alleviate that. I believe it's the better option if you want to use WCT in near-real time.

What makes the use of these files possible is that on the NWS servers, they include a file: "sn.last", which will always hold the newest data for a given product. So each WCTFAV file can pull up the most recent data for a given product. In addition, to cut-down on quantity of files, I included the airport TDWR Radar data along with a nearby WSR-88d location.
