Unofficial 2021 HURDAT2 for the Atlantic Basin

* This page is NOT endorsed nor supported by NOAA or its subsidiaries, including the National Hurricane Center. *

  • Updated 10 NOV 2021

* Information presented here is retrieved from current-season 'best-track' (https://ftp.nhc.noaa.gov/atcf/btk/) data. Because landfall-data are not recorded in the best-track data, they are compiled using the advisory archive (https://ftp.nhc.noaa.gov/atcf/adv) and GIS software.

For such landfall data, wind-extent information is currently not included. As such, 'null' space-fillers (-999) are used where applicable.

Precision is not the aim of landfall inclusion (especially the interpolated ones) but were pursued to provide homogeneity and temporary inclusion with the official HURDAT2 record. The official record is usually updated during the Spring of each year, prior to the start of Hurricane Season.

* Below you'll find some stats and ranks for the current and other season(s). The stats are valid thru the dates listed.

My primary purpose for this is for use in my hurdat2parser program by concatenating it with the official data. One can then compare the current-season to previous years using a supplied function that ranks partial seasons.

Feel free to use this output in any way you'd like. My only request of it is that if used publicly, you must provide the disclaimer that the data, though mostly retrieved from NOAA, is not NOAA-official, published, nor generated.
