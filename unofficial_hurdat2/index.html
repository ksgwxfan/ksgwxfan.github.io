* Information presented here is formulated from current-season 'best-track' (https://ftp.nhc.noaa.gov/atcf/btk/) data.

			
Because landfalls are not recorded in the best-track data, they are compiled using the advisory archive (https://ftp.nhc.noaa.gov/atcf/adv) and track observation (estimating location and time).

			
For such landfall data, wind-extent information is currently not included. As such, 'null' space-fillers (-999) are used where applicable.

			
The aim of landfall inclusion is homogeneity with the offical Hurdat2 record.

			

		


		

			

			
The primary purpose for the current-season Hurdat2 is for concatenation with the NHC-released Hurdat2, which is usually updated during the Spring of each year prior to the start of Hurricane Season. Because of this, the latest hurdat2 data wouldn't include data from the current season. In addition it is for use in my hurdat2parser program, allowing comparison of the current-season to previous years using a supplied function that ranks partial seasons (it's how the above stats are generated).


			
Feel free to use this output in any way you'd like. My only request of it is that if used publicly, you must provide the disclaimer that the data is not NOAA/NHC official.
