* Aug 22, 2022 Update :: Data for the East/Central Pacific Basins will now be included.

			
* Information presented here is formulated from current-season 'best-track' (https://ftp.nhc.noaa.gov/atcf/btk/) data.

			
Because landfall-data are not recorded in the best-track data, they are compiled using the advisory archive (https://ftp.nhc.noaa.gov/atcf/adv) and observing the track (estimating location and time).

			
For such landfall data, wind-extent information is currently not included. As such, 'null' space-fillers (-999) are used where applicable.

			
Precision is not the aim of landfall inclusion (especially the interpolated ones) but were pursued to provide homogeneity with the actual Hurdat2 record.

			

		


		

			

			
The primary purpose for the current-season Hurdat2 is for concatenation with the NHC-released Hurdat2, which is usually updated during the Spring of each year, prior to the start of Hurricane Season, and as such would not include current-season data. In addition it is for use in my hurdat2parser program, allowing comparison of the current-season to previous years using a supplied function that ranks partial seasons (it's how the above stats are generated).


			
Feel free to use this output in any way you'd like. My only request of it is that if used publicly, you must provide the disclaimer that the data, though aretrieved from NOAA/NHC, is not NOAA-official, nor generated.
