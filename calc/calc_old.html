Almost all you see here is HTML/CSS/Javascript; no SVG's were used. I used Python to create code for the star-field in my space theme. I hope you bookmark this calculator, come back often, and that it is of use to you. If you discover something that isn't working right, GOOD! Let me know, and I'll tackle it as soon as I can. This has been a great exercise in trying to account for every kind of problem that can arise. So if it's broken, it means it can be improved! I have found an appreciation for the creation of the most basic calculators. Some items I had to account for include:

  • Preventing 'division by zero' attempts
  • Preventing successive zero entries at the start of a number
  • Display-limit of characters so numbers couldn't extend beyond the virtual display
    • This included defining situations where I wanted scientific-notation or wanted to limit the output
  • Ignoring successive operator entries (sorry if you accidentally hit the wrong button)
