THIS GAME DOES HAVE SOUND! My dad used to work extensively at Radio Shack, so we'd always have bunches of radio-shack products and toys. This is a program that is similar to the Pocket-Repeat. I almost scrapped this project due to difficulty navigating javascript timing issues. Persistence paid off! The best way I can describe my method is a pseudo-loop controller, which allowed me to use the timing functions I wanted. I was stoked after that. The project was fairly smooth sailing. Sound was a last minute add-in. I found a youtube review of the pocket-repeat and used GoldWave to mimic the sounds of the button-presses. Sound was something I had not messed with in HTML5/CSS/Javascript, so I quickly became confused. Luckily, I calmed down a little bit and found a site that went step-by-step on how to use it. In hind-sight, I could've used W3schools, but sometimes I get confused and feel some explanations are vague (but it's mostly impatience on my part). I won't allow the use of this creation elsewhere, but please contact me if you want to use some of the code or some clarification/how-to to help you on your project (commercial or not) I'll try to accommodate you.
