Welcome to MineSweep, a minesweeper clone. The goal is to preferably isolate the locations of all of the mines in the field. Click a cell to test it. If there are any mines in its proximity, the quantity will appear. Press 'F' to toggle flag mode. This will help mark the spaces you feel have mines. This project took a long time but after a many-months break. I generally complicate things too much. I've often felt that programming is like a play. The audience (user) only sees the stage production (program). You don't see the hours of practice(troubleshooting) or the crew in the background (code structure; if/else statements) making the on-stage magic possible. You may see the script (the source code), but you wouldn't see the time it took to write the play. I hope you enjoy the game. If you want to use the game to modify or place on your site, feel free, but attribution either in your website code or on the page is all I require. If you simply like a portion of the code (a function spurring your interest/idea), don't worry about attribution. Thanks.
